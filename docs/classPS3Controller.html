◆ controller_state

uint8_t PS3Controller::controller_state[STATE_ARRAY_SIZE]
private

8-bit integer array that stores the controller state.

Stores the controller's current inputs in an array whose size is defined at the top of the file by STATE_ARRAY_SIZE. The values are stored as 8-bit integers, which, in this case, come directly from the controller.
